Detailed Description

Header for a base class for small program configuration files.

This file contains headers for a base class used to create a parser for really simple configuration files which are to be kept on an SD cards or similar devices. The SD card (or other device) is expected to be used in a small microcontroller based data logger or similar device. This base class provides generic configuration file reading functions; it is expected that descendents will be designed to read specific files in specific projects.

  • 12-27-2009 JRR Added tools for reading booleans and integers

Macro Definition Documentation

#define CFGF_DBG (   line)    if (p_serial) *p_serial << line

Debugging printout for the A/D converter.

If CFGF_DBG is defined as a statement which prints something, this function macro writes debugging information to the serial device pointed to by p_serial, if that serial device pointer points to an actual device. If the CFGF_DBG is defined as nothing, this macro evaluates to nothing, turning debugging statements off.

Parameters
lineA line of printable stuff, separated by << operators, which follows the first << in a printing line
