51 /**
52  *
53  * @section cmd_codes "Command Codes"
54  * The following structure is used for commands sent from the PC to the
55  * microcontroller:
56  * | 1 byte | 0-4 bytes | 1 byte |
57  * |----------|-----------|----------|
58  * | command | data | checksum |
59  * Commands:
60  * \li 0 - F : No data : Send A/D reading from one channel, 0 through 15
61  * \li More to come...
62  *
63  * @section data_packets "Data Packets"
64  * The following structure is used for data sent by the microcontroller to
65  * the PC:
66  * \li Packet header: Begin with a byte which is 0xAA
67  * \li Time: 32-bit number containing milliseconds since program startup
68  * \li Time per point: 32-bit number containing microseconds per reading set
69  * \li Active channels: 16-bit bitmask, active channels being 1
70  * \li A/D data: 16-bit number; 4 MSB's 0000; other 12 bits have the data
71  * \li CRC16 for packet
72  * \li Others...?
73  */
