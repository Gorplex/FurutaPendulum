Detailed Description

Task which acquires PolyDAQ data at precise time intervals.

This task acquires data. It is run at a high priority so that data can be taken at precise time intervals. The data is then put into queues so that other tasks can print the data through a serial port, store the data onto an SD card, or do whatever else is needed.
