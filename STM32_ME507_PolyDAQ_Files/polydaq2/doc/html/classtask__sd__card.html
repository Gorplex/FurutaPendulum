Detailed Description

Task which saves data to an SD card.

This task saves data to an SD card. The data is transferred to this task as a stream of bytes in a queue.

void task_sd_card::run ( void  )

The run method that saves data to an SD card.

This run method waits for an SD card to be inserted in the socket. When a card is found in the socket, it opens a data file using an automatically generated, sequenced name. Then data which arrives in the data queue is saved to the card as soon as the data arrives.

void task_sd_card::write_header ( emstream *  p_ser_dev)

Write a header showing data items to be logged to a serial device.

This method writes a header line which shows the name of each of the items to be logged. Usually the logging is done to an SD card, and this line is a header for the logged data. The logger configuration object, which is pointed to by the shared variable p_logger_config, is used to get the data headers.
