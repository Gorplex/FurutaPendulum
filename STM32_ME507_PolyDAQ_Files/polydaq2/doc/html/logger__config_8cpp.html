Detailed Description

Source code for a parser for data logging configuration files.

This file contains source code for a parser for simple configuration files that set up channels on a simple data logger using a PolyDAQ 2 data acquisition board.

  • 05-23-2011 JRR Modified for use with interrupt timed data acquisition
