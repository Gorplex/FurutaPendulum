Detailed Description

Task which prints out characters in the main print queue.

This task prints characters which are in the main print queue which is pointed to be p_main_text_queue. If the queue is empty, this task is blocked by its getchar() method and does nothing.

void task_print::run ( void  )

The run method that runs the main queue printing thread.

This method checks for characters in the main print queue whose address is at p_main_text_queue and prints any characters found.

Returns
This method should never return; if it hypothetically did, it would return an OS message of zero.
